# Authentic Goal Setting

オーセンティックコーチングで語られる「現状の外側のゴール設定」を参考にした、個人開発の静的セルフワークページです。

## 注意

このページは公式教材ではありません。オーセンティックコーチング、苫米地式コーチング、または関連書籍の内容を完全または正確に再現できている保証はありません。専門的なコーチング、医療、心理、法律、投資などの助言でもありません。

## Local Preview

```bash
python3 -m http.server 8083 --directory authentic-coaching-goal-page
```

Open <http://localhost:8083>.

## Features

- 領域別のゴール候補入力
- want-to、現状の外側、社会性、臨場感のスライダー評価
- ゴール文の自動整理
- 次に深掘りする問いと日次キューの生成
